# contexte
L'application est un site pour une communauté gaming, il propose les fonctionnalités suivante:
- Streamer : peut suivre des streamers de la communauté avec des lecteurs vidéo accessible sur le site ainsi que le chat du stream
- Guides : le site peux proposer des guide jeux qui peuvent être versionné avec un format spécifique pour world of warcraft ainsi que des guides autre que les jeux
- créations : exposer des créations artistique ou en programmation.

# stack technique
- **languages** : PHP (dernière version stable), MySQL, typescript, Vue.js (dernière version stable)
- **Framework** : Laravel (dernière version stable) 
- **Package** : Inertia.js, Filament (pour le back office)
- **Styling** : TailwindCSS avec Flowbytes

# Déploiement
Utilisation de Docker via Laravel Sail 

# Structure du projet
- app/ : fichiers Laravel
- resources/js :  fichier typescript et Vue.js
